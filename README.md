📋 Payroll Management System v20.0
Welcome to the Payroll Management System. This tool is designed for consultants and managers to track employee work hours, manage multiple branches, and calculate payouts efficiently without needing a complex backend.

🚀 Getting Started
Access the Tool: Simply open the link provided in this repository.

Employee Setup: Type the employee's name and branch. The tool will remember them for future entries.

Log Work: Enter the date and shift times. You can toggle between 24-Hour and 12-Hour formats.

Save Data: Click "Save / Update Log". Your data is stored locally in your browser.

🛠 Key Features
Smart Import (De-duplication): Import your previously exported CSV files. The tool will automatically update existing records and add new ones without creating duplicates.

Bulk Profile Updates: Need to give someone a raise or move them to a new branch? Use the Global Profile Update to change all historical records at once.

Punch Card Preview: Upload a PDF of a physical punch card to view it side-by-side while you enter data.

Duplicate Row: Click any row in the log to show a "Duplicate" button—perfect for employees who work the same shifts every day.

🔒 Privacy & Data Safety
Your Data stays with YOU: This is a "Client-Side" application. No data is sent to a server. Everything is saved in your browser's local storage.

Backup Often: Because data is local, clearing your browser cache may delete your logs. Always export your data to a CSV file at the end of each week.
